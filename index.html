    Panduan & Tentang

    Cara Menggunakan

    1. Pilih dzikir dari dropdown
    2. Atur target jumlah dzikir
    3. Klik pada lingkaran tasbih
    4. Reset untuk mengulang hitungan

    Tips Berdzikir

    • Lakukan dengan khusyuk
    • Rutinkan setiap pagi dan petang
    • Fokuskan hati saat berdzikir

    Tentang TasbihKu

    Aplikasi penghitung dzikir digital untuk membantu ibadah harian.

    Versi: 2.1.0

    Pertanyaan Umum

    Bagaimana cara reset hitungan?

    Gunakan tombol reset di bagian bawah counter.

    Apakah data tersimpan?

    Ya, data disimpan secara lokal di perangkat Anda.

    Bisakah mengubah target?

    Ya, melalui input target di bagian atas.
